# Job Portal

A comprehensive job portal application built with React for the frontend and Node.js for the backend, featuring job search, user authentication, and application tracking.

---

## Features

- Job seekers can:
  - Search and filter jobs by location, role, and salary.
  - Apply for jobs and upload resumes.
- Recruiters can:
  - Post new jobs.
  - View applications and manage job postings.

---

## Tech Stack

### Frontend:
- **React**
- **Redux Toolkit** (for state management)
- **React-Bootstrap** (for styling)

### Backend:
- **Node.js** (Express framework)
- **MongoDB** (for database management)
- **Cloudinary** (for file storage like resumes and profile pictures)
- **JWT Authentication**

### Deployment:
- Hosted on [Vercel].
